# Assessments

## Course Assessments

### 1. ROS 2 Package Development Project
- **Objective**: Demonstrate proficiency in ROS 2 development
- **Deliverables**: 
  - Functional ROS 2 package
  - Documentation
  - Demonstration video

### 2. Gazebo Simulation Implementation
- **Objective**: Create and simulate a robot in Gazebo
- **Deliverables**:
  - Robot model (URDF/SDF)
  - Simulation world
  - Sensor integration

### 3. Isaac-Based Perception Pipeline
- **Objective**: Implement perception using NVIDIA Isaac
- **Deliverables**:
  - VSLAM implementation
  - Object detection pipeline
  - Performance analysis

### 4. Capstone: Simulated Humanoid Robot with Conversational AI
- **Objective**: Complete end-to-end humanoid robot system
- **Deliverables**:
  - Working simulation
  - Voice command interface
  - Object manipulation
  - Complete documentation
  - Demo video

## Grading Criteria

- **Functionality** (40%): Does it work as specified?
- **Code Quality** (20%): Clean, documented, maintainable code
- **Documentation** (20%): Clear documentation and comments
- **Innovation** (10%): Creative solutions and extensions
- **Presentation** (10%): Clear demonstration and explanation
